無人載具科技創新實驗條例

國際間發展無人載具，主要以無人車輛為發展重點，其中又可區分為商業運輸車輛與個人私有車輛。

商業運輸車輛發展目前以自動駕駛化為主，如自動駕駛巴士、自動駕駛貨車，利用自動駕駛車輛降低營運成本並提升運輸效率。

個人私有車輛發展目前以駕駛輔助為主，透過系統輔助降低駕駛人行車疲勞並提升行車安全。

國際間各類車輛為求符合實際道路使用需求，已陸續走出實驗場域，於各地區進行公共道路實地測試。為了解一般大眾對自駕測試的想像及感受，歡迎大家提出各種想法和意見。
